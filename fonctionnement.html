Fonctionnement du site

1. Architecture technique

Le site de l'ESF Le Faucon est conçu comme une application web hybride utilisant :

  • Interface (Frontend) : HTML5, CSS3 progressif et JavaScript (ES6+).
  • Serveur (Backend) : Un environnement Node.js avec le framework Express pour la gestion des sessions et des API.
  • Base de données : MySQL pour le stockage sécurisé des informations académiques et administratives.

2. Compatibilité

Pour une expérience optimale, nous recommandons l'utilisation des dernières versions des navigateurs suivants :

  • Google Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge

3. Cookies et Sessions

Notre site utilise des cookies techniques strictement nécessaires au fonctionnement de l'espace membre (Espace Étudiant, Administration). Ces cookies permettent de maintenir votre session connectée de manière sécurisée. Aucun cookie publicitaire de tiers n'est utilisé sur ce site.

4. Crédits

Design et développement réalisés par le pôle informatique de l'École Supérieure Le Faucon. Tous les visuels, sauf mention contraire, sont la propriété de l'école ou utilisés sous licence Creative Commons.
